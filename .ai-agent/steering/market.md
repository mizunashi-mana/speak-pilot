# 市場分析

## 市場概要

macOS 向け音声入力ソリューション市場。キーボード入力に代わる、あるいは補完する入力手段としての音声入力。

## ターゲットセグメント

- **マルチタスクワーカー**: 手がふさがっている状況でもテキスト入力したいユーザー
- **キーボード入力を減らしたいユーザー**: 長時間タイピングによる負担を軽減したい人
- **音声入力をデフォルトの入力手段にしたいユーザー**: キーボードに依存しないワークフローを目指す人

## 競合分析

| プロダクト | 特徴 | 制約 |
|-----------|------|------|
| macOS 標準音声入力 | OS 組み込み、無料 | 認識精度に限界、カスタマイズ性低い |
| Aqua Voice | 高精度（技術用語97%精度）、50ms未満で起動、Mac/Windows対応、独自モデル Avalon 搭載 | クラウド依存、有料サブスクリプション |
| Superwhisper | macOS ネイティブ、Whisper ベース、オフライン動作、Apple Silicon 最適化 | 有料（月$8.49〜、買い切り$249.99） |
| Whisper.cpp / MLX Whisper | 高精度、ローカル実行、オープンソース | 単体アプリではない、統合が必要 |
| Google 音声入力 | 高精度、多言語対応 | クラウド依存、プライバシー懸念 |

## 差別化ポイント

- macOS ネイティブアプリとしてシステム全体に統合
- 音声入力を「補助的な機能」ではなく「デフォルトの入力インターフェース」として設計
- ローカル音声認識によるプライバシー重視（MLX Whisper）
- マルチタスク支援に特化した UX
- オープンソース / 自前運用可能（クラウドサービスへの依存なし）

## 市場動向

- ローカル LLM / ローカル AI 処理への関心の高まり
- Apple Silicon による高速なオンデバイス推論の実現
- Whisper モデルの軽量化・高速化の進展（MLX、distil-whisper 等）
- 音声インターフェースの一般化（AI アシスタント普及の影響）
- Aqua Voice、Superwhisper など macOS 向け音声入力アプリの活発な開発
